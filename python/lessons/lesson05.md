# Lesson 05 - Installing PyCharm

## Lesson Summary

This lesson introduced PyCharm as a Python IDE and explained how to configure a development environment. It also showed an alternative for programming on Android using QPython3, making it possible to practice even without access to a computer.

## What I Learned

- What an IDE is and why it improves the programming experience.
- How to install and configure PyCharm.
- That it is possible to write and run Python code on an Android device.
- The importance of practicing consistently through exercises.

## My Reflection

Although the course introduced PyCharm, I decided to continue using Visual Studio Code because I find its interface more intuitive and I believe it is a tool I will most likely use in my professional career.

The part that caught my attention the most was learning that I can also study Python using my phone.

During school vacations, I usually travel to my father's house and I cannot take my laptop with me. Knowing that I can continue practicing through my phone means I don't have to interrupt my learning.

## Personal Note

This lesson reminded me that consistency is more important than the place where I study.

As long as I keep practicing, I can continue learning whether I'm using my computer or just my phone.

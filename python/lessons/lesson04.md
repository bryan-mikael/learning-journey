# Lesson 04 - First Commands in Python

## Lesson Summary

In this lesson, I learned the first fundamental Python commands and started writing my first interactive programs.

The lesson covered:

- `print()` to display information on the screen.
- `input()` to receive data from the user.
- Variables and how they store information.
- The difference between Interactive Mode and Script Mode.
- Practical programming challenges.

## What I Learned

- How to display messages using `print()`.
- How to receive user input using `input()`.
- How variables make programs more organized and easier to understand.
- The difference between testing code in Interactive Mode and creating `.py` files in Script Mode.
- How to build simple interactive programs.

## My Reflection

The most important concept I learned in this lesson was variables.

Understanding how variables work helped me understand the logic behind Python and how programs are organized.

The practical challenges were the part that helped me the most. Instead of only watching the lesson, I immediately applied what I had learned, making the concepts much easier to understand.

One of the biggest lessons I learned was that programming is not about memorizing syntax. It is about understanding the logic behind the code. Once the logic makes sense, the syntax becomes much more natural.

## Personal Note

Without the practical exercises, I don't think I would have learned as efficiently.

The challenges were essential because they allowed me to practice immediately after learning each concept.

This lesson was the first time I felt that I was actually programming instead of simply watching someone else write code.
